## PREPARACIÓN DE LOS ARCHIVOS DE ENTRADA
**PLANTA**
Con el archivo de planta obtener los datos de los horarios y materias del semestre.
Estas serán las opciones que disponemos para la creación de las tribus de horarios para los alumnos.

(1) Descargar el archivo de la planta del ciclo actual en formato .csv (esto para evitar conflictos con las fórmulas)
(2) Abrir un nuevo libro de excel y en la pestaña de Datos seleccionar obtener y transformar datos desde csv con el formato UTF-8
(3) Borrar el formato con la macro LIMPIAR REPORTE y quitar las filas de encabezado.
(4) Guardar el archivo en la carpeta raw (para programa) correspondiente al ciclo actual

**PLAN ACADÉMICO*
Revisar la oferta de materias con SAE:
- Materias que se ofertan según el ciclo actual (enero/agosto)
- Materias según el plan para la carrera (revisar directamente con el coordinador de carrera)
- Editar el plan para que tenga los nombres de materia de la SC (Sección combinada)
